## 📖 简介 (Introduction)

**LifeQuest** 是一款硬核的目标管理工具，专为那些厌倦了枯燥 To-Do List 的奋斗者设计。

我们不只是记录待办事项，我们将你的备考、工作、锻炼转化为一场**史诗级的 RPG 战役**。在这里，你通过完成现实任务赚取经验（XP）和金币（Gold），提升你的四维属性，并与一个全天候成长的“宿敌”进行竞速。

**v3.0 "觉醒版"** 带来了全新的连击系统、循环任务机制以及更智能的对手。

## 🎮 游戏机制 (Game Mechanics)

### 📊 1. 四维属性雷达 (Attributes Radar)
你不再是苍白的“完成者”，你是拥有属性的英雄。每个任务都可以绑定一种属性：
*   **Perception (感知)**：如听力训练、观察练习。
*   **Insight (洞察)**：如阅读、分析报告。
*   **Logic (逻辑)**：如写代码、数学题、写作。
*   **Charisma (魅力)**：如口语练习、演讲、社交。
> *随着任务完成，你的雷达图会动态扩张，直观展示你的长板与短板。*

### 😈 2. 智能宿敌系统 (Rival System)
这是 LifeQuest 的核心。你有一个名为“宿敌”的影子对手：
*   **全天候成长**：即使你关闭软件，他也在离线挂机升级。
*   **随机进化**：每隔 10 分钟，系统会随机判定宿敌是否“努力”了一把。
*   **难度分级**：从 "摸鱼怪 (0.5x)" 到 "神 (2.0x)"，选择适合你的挑战难度。
*   **竞速压力**：右上角的红色进度条时刻提醒你——**他不睡觉，你敢睡吗？**

### 💰 3. 经济系统与战利品 (Economy & Shop)
自律需要正反馈。
*   **赚取赏金**：完成任务获得金币。
*   **战利品商店**：你自己定义商品的奖励（如：喝奶茶、打游戏 1 小时、买新手办）。
*   **消费**：用辛苦赚来的金币购买这些欲望。这让每一次“玩乐”都变得心安理得。

### 🔄 4. 任务体系 (Quest Board)
*   **任务类型**：支持 **一次性任务** 和 **长期循环任务**。
*   **生效日**：你可以设置循环任务只在“周一、三、五”出现，周末自动休息。
*   **过期惩罚**：今日事今日毕。如果循环任务当天未完成，第二天它会被标记为 **[已过期]** 并计入黑历史，这是你战绩上的污点。
*   **连击火焰 (Streak)**：保持每日活跃，点燃连击火焰！连击天数越高，获得的经验金币加成越多（最高 +50%）。

## 🚀 下载与安装 (Installation)

### 对于所有用户 (All Users)
LifeQuest 现已打包为单文件程序，无需安装 Python。

1. 前往 [Releases 页面](../../releases) 下载最新的 `LifeQuest_V3.1.exe`。
2. 将它放在你喜欢的任何地方（桌面、D盘等）。
3. **直接运行**。

> **注意**：首次运行时，软件会自动在你的系统用户目录下创建存档数据库。

## 💾 数据迁移指南 (Migration Guide)

**如果你是从 v2.0 或更早版本升级的老用户：**

为了保持桌面整洁，v3.1 不再在 EXE 同级目录生成文件。请按以下步骤迁移你的老存档：

1. 下载并运行一次 v3.1 新版，然后关闭它（让系统自动建立新文件夹）。
2. 在键盘上按 `Win + R`，输入 `%APPDATA%\LifeQuest` 并回车。
3. 找到你旧版本文件夹里的 `lifequest.db`（存档）和 `config.json`（配置）。
4. 将旧文件**复制并粘贴**到第 2 步打开的文件夹中，覆盖新文件。
5. 重启软件，你的等级、金币和宿敌就全部回来了！
